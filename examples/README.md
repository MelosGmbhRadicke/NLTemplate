# export-generator #

A demonstrator application

## Dependencies ##

- cmake
- LaTeX
 * pdfladex
 * texlive-pgfplots


## Build ##

> mkdir ./build
> cd ./build
> cmake -G "Unix Makefiles"  ../
> make
